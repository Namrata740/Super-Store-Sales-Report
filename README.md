# 📊 Super Store Sales Analytics Dashboard

## 🔍 Overview  
This project showcases an interactive Power BI dashboard analyzing Super Store sales data. It highlights key metrics, trends, and business insights for better decision-making.

## 📸 Preview  
*([Add dashboard screenshot here](https://github.com/Namrata740/Super-Store-Sales-Report/blob/main/Super%20Store%20Sales%20Dashboard.png))*

## 📈 Key Highlights  
- 💰 Sales: **1.6M+**  
- 📦 Orders: **22K+**  
- 📊 Profit: **175K+**  
- 🚚 Avg Ship Days: **4**  

- 👥 Consumer segment leads (~48%)  
- 🌍 West region is top-performing  
- 💳 COD is the most used payment mode  
- 📱 Phones are the top-selling sub-category  
- 📅 Peak sales in **Q4 (Oct–Dec)**  

## 🛠️ Tools Used  
- Power BI  
- Excel / CSV  
- DAX  

## 🚀 How to Use  
Download the `.pbix` file and open it in Power BI Desktop to explore the dashboard.

## 💡 Insights  
Focus on boosting online payments, improving low-performing regions, and leveraging high-demand products for growth.
